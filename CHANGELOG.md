# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-07

### Added

- **Text Transform**: UPPER CASE, lower case, camelCase, PascalCase, snake_case, kebab-case with multi-cursor support
- **Encode/Decode**: Base64 Encode/Decode (with input validation), URL Encode/Decode
- **JSON Tools**: Format JSON (2-space indent) and Minify JSON with pre-validation
- **Code Generation**: UUID v4, Lorem Ipsum (1-5 paragraphs), Insert Date with multi-step wizard
- **Regex Tester**: Interactive WebView panel with live pattern matching, CSP-protected
- **Activity Bar**: Dedicated sidebar with categorized Tools list and operation History
- **History**: Track all operations with relative time display (e.g. "3 min. ago")
- **File Watcher**: Monitor `.quickutilsrc` / `.quickutilsrc.json` for project configuration changes
- **API Integration**: Secure API key storage (OS keychain) with auto-retry and progress reporting
- **Localization**: English and Japanese (l10n) support
- **Configuration**: Log level, date format, history size, auto-retry settings
- **Context Menu**: Right-click submenu for Transform Text, Format JSON, Generate UUID
- **Keyboard Shortcut**: `Ctrl+Shift+T` / `Cmd+Shift+T` for Transform Text
- **Status Bar**: Quick Utils indicator with one-click access to Transform Text
